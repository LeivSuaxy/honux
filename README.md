# Honux

Honux is a complete system for home automation and data collection. This repository contains all the code and documentation related to the software, firmware, and infrastructure developed for self-hosting in my own home.

Each project includes its own `README.md` file with detailed instructions and documentation.

## Development Principles

The technologies used in this project are primarily chosen to ensure high performance and low resource consumption. Most components are designed to run on Linux-based operating systems, while the firmware may vary depending on the available electronic components. Specific firmware and hardware details are documented within each individual project.

## Repository Structure

The repository is organized into the following directories:

```
├── docs # All documentation for the project.
├── honux-core # The core software for the Honux system, is the main system.
├── honux-dashboard # The dashboard for the Honux system. Panel admin and monitoring.
├── honux-devices # All firmware code for Arduinos, ESP32 and other open-source hardware devices and sensors.
├── honux-libs # Shared libraries and utilities used across the Honux projects.
├── honux-mcps # Model Context Protocols servers used for AI Models to perform actions on the system.
```

This documentation is not finished yet and is currently a work in progress.
